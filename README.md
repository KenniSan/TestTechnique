# Mini Web App

Application simple avec frontend et backend séparés.

## Stack
- Frontend : HTML, CSS, JavaScript
- Backend : Node.js + Express

## Installation

### Backend
cd backend
npm install
npm start

### Frontend
Ouvrir index.html dans un navigateur
Méthode 1:
    cd frontend
    python3 -m http.server 5500

Méthode 2:
    Ouvrir index.html dans un navigateur

## Fonctionnalités
- Ajouter un élément
- Afficher la liste
